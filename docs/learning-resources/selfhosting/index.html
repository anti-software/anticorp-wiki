Learning resources #

  • vagrant-vmware-box: How to create VMware vagrant box
  • roll.urown.net: How to roll your own private self-hosted internet services

  • Red Team Infrastructure Wiki: Wiki to collect Red Team infrastructure hardening resources

  • focalboard: Focalboard is an open source, self-hosted alternative to Trello, Notion, and Asana.
Design #
  • penpot: The Open-Source design & prototyping platform

Transmission #

  • logseq: A privacy-first, open-source platform for knowledge management and collaboration

  • cryptpad: Collaborative office suite, end-to-end encrypted and open-source
  • hugo-book: Hugo documentation theme as simple as plain book

  • onionpipe: It forwards ports on the local host to remote Onion addresses as Tor hidden services and vice-versa.
Development #
  • onedev: Self-hosted Git Server with CI/CD and Kanban
Archival #
  • wallabag: wallabag is a self hostable application for saving web pages: Save and classify articles. Read them later. Freely.

  • vaultwarden: Unofficial Bitwarden compatible server written in Rust, formerly known as bitwarden_rs
  • TheHive: A scalable, open source and free security incident response platform

  • ntfy: Send push notifications to your phone or desktop using PUT/POST
  • uptime-kuma: A fancy self-hosted monitoring tool
  • awesome-selfhosted: A list of Free Software network services and web applications which can be hosted on your own servers
